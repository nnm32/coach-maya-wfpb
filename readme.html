Coach Maya — Whole-Food, Plant-Based Coach

This repository contains a single-page web app with a speaking avatar (Coach Maya), password login, onboarding (name + one-line motivation), and daily typed check-ins for four WFPB goals. Data is stored locally in the browser and is erased when site data/history is cleared. No backend and no email are used in this build.

Files

  • index.html — the complete app (avatar, login, onboarding, check-ins, streak, history, TTS).
  • readme.html — this documentation page.
  • worker.html — placeholder showing how a Worker would be added later (not required).
  • wrangler.html — placeholder showing a future deployment config (not required).

How to run

  1. Open index.html directly in your browser (double-click).
  2. Create a username + password → enter your first name + motivation.
  3. Use the Daily Check-In. Maya will speak if you toggle Voice On.

Notes

  • All data is stored in localStorage/sessionStorage on the device.
  • To reset, clear the site’s storage or click Create/Reset with a new password.
  • The avatar is a CSS/HTML drawing designed to feel young, Hispanic, and motherly.
