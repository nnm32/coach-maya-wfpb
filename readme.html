Coach Maya — Whole-Food, Plant-Based Coach

Single-page app with a photoreal video avatar, natural US-English voice (best available on device), password login, onboarding (name, one-line motivation, email/frequency/time), daily typed check-ins for four WFPB goals, streak, 10-day history, and EmailJS reminders (client-only; fires while the page is open + catch-up on next visit).

Files

  • index.html — the complete app (avatar, voice, login, onboarding, check-ins, EmailJS reminders).
  • readme.html — this documentation page.
  • worker.html — placeholder; not used in this client-only build.
  • wrangler.html — placeholder; not used in this client-only build.

Requirements

  1. Place a short looping video named maya_idle.mp4 alongside index.html. (Blink/smile portrait, 3–8s.)
  2. Optional: create a free EmailJS account and paste your EMAILJS_PUBLIC_KEY, SERVICE_ID, and TEMPLATE_ID in index.html.
  3. Open index.html or host via GitHub Pages.

Notes

  • All progress stays on the device (localStorage/sessionStorage). Clearing site data erases it (by design).
  • Client-only emails cannot send while the browser is completely closed; for 24/7 delivery, add a tiny Worker later.
  • Fallback CSS avatar appears if the video can’t load.
